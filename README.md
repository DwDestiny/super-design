## English

### Overview
Super Design is a design workflow skill that helps generate UI prototypes, visual designs, and HTML mockups with a lightweight local dashboard for reviewing iterations.

### What it includes
- Design workflow prompts and templates
- Dashboard for viewing generated HTML pages in a canvas-style board
- Utility scripts for local preview

### Use cases
- Rapid UI/UX prototyping
- Web/app/tool interface design
- Design iteration management and review

### Quick start
1. Place this folder under your Codex skills directory (e.g. `~/.codex/skills/`).
2. Load the skill by name: `super-design`.
3. Generate HTML design pages.
4. Start the local board server to review pages:
   - Run the provided script in your project: `./.superdesign/design_iterations/start_board.sh`
   - Open the local URL printed in the terminal.

## 中文

### 概述
Super Design 是一个用于 UI 原型、视觉设计与 HTML 设计稿输出的工作流技能包，并提供一个轻量本地看板用于查看设计迭代。

### 包含内容
- 设计工作流提示词与模板
- 用于查看 HTML 设计稿的看板（画布式）
- 本地预览相关脚本

### 适用场景
- 快速 UI/UX 原型设计
- 网站/工具/App 设计
- 设计迭代管理与评审

### 快速开始
1. 将本文件夹放入 Codex 的 skills 目录（例如 `~/.codex/skills/`）。
2. 通过技能名称加载：`super-design`。
3. 生成 HTML 设计稿页面。
4. 启动本地看板进行查看：
   - 在项目中运行：`./.superdesign/design_iterations/start_board.sh`
   - 打开终端输出的本地地址。
