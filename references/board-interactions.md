# Board Interactions (UX)

## Layout
- Left: group list (design families)
- Center: grid cards for designs
- Right: preview panel (iframe)

## Core interactions
- Select a group to filter cards
- Search by filename
- Click a card to preview

## Version display
- Show variant (A/B/C) and version number
- Show last updated timestamp

## Recommended improvements
- Add a compare mode (2-up preview)
- Add filters: variant, time, tag
- Add a "latest only" toggle
